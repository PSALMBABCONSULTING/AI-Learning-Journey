# Lesson 01: Supervised vs Unsupervised Learning

## Supervised Learning

Supervised learning is a type of machine learning where the model learns from data that already has known answers or labels.

For example, if we have historical data showing which residents had falls and which did not, the model can learn patterns that help predict future fall risk.

## Unsupervised Learning

Unsupervised learning is a type of machine learning where the model looks for hidden patterns in data without being given known answers.

For example, it can group similar patients, units, or facilities based on patterns in their data.

## Key Difference

Supervised learning uses labeled data with known outcomes, while unsupervised learning finds patterns in data without known outcomes.
