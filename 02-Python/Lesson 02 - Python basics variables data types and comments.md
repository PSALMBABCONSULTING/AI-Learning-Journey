# Lesson 02: Variables, Data Types, and Comments

## What is a Variable?

A variable is a name used to store a value in Python.

For example:

```python
patient_group = "Older Adults"
total_patients = 150
fall_incidents = 12
```

Variables help store information that can be reused in a program.


## Common Python Data Types

| Data Type | Meaning | Example |
|---|---|---|
| String | Text | `"Fall Risk Dashboard"` |
| Integer | Whole number | `150` |
| Float | Decimal number | `94.5` |
| Boolean | True or False | `True` |
| List | Collection of values | `["Falls", "Readmissions", "Infections"]` |

## String

A string is text. Strings are written inside quotation marks.

```python
dashboard_name = "Fall Risk Dashboard"
care_area = "Long-Term Care"
```

## Integer

An integer is a whole number.

```python
total_patients = 150
fall_incidents = 12
```

## Float

A float is a number with decimals.

```python
compliance_rate = 94.5
average_age = 78.6
```

## Boolean

A Boolean value is either `True` or `False`.

```python
training_completed = True
high_risk_patient = False
```

## List

A list stores multiple values.

```python
quality_indicators = ["Falls", "Readmissions", "Infections", "Medication Errors"]
```

## Comments

Comments are notes in Python code. They start with `#`.

```python
# This stores the name of the dashboard
dashboard_name = "Fall Risk Dashboard"

# This stores the number of fall incidents
fall_incidents = 12
```

## Healthcare Analytics Example

Python variables can be used to store healthcare analytics values such as dashboard name, care area, reporting period, patient count, fall incidents, readmission count, compliance rate, and quality indicators.

Example:

```python
dashboard_name = "Fall Risk Dashboard"
care_area = "Long-Term Care"
total_patients = 150
fall_incidents = 12
compliance_rate = 94.5

print(dashboard_name)
print(care_area)
print(total_patients)
print(fall_incidents)
print(compliance_rate)
```
