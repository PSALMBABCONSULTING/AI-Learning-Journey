## Lesson 01: Why Python Matters for Healthcare Analytics and AI

Python is important because it helps you move beyond dashboards into automation, data cleaning, machine learning, and AI workflows.

For your career, Python can help you:

- Clean healthcare data
- Automate Excel and CSV tasks
- Connect to APIs
- Analyze large datasets
- Build machine learning models
- Prepare data for Power BI
- Create reusable analytics scripts
- Support AI and predictive analytics projects

The learning path inside 02-Python should be:

- Why Python for healthcare analytics
- Python basics: variables, data types, and comments
- Lists and dictionaries
- Conditions and loops
- Functions
- Pandas for data analysis
- Reading Excel and CSV files
- Cleaning healthcare-style data
- Basic charts
